钻井破岩参数换算

面向钻井破岩计算的常用单位换算页。当前覆盖机械钻速、转速、钻头直径、井底面积、每转进尺和机械比能估算。输入统一转成 SI 单位后再计算，方便后续接入数值模型或实验数据表。

机械钻速 ROP

常见记录单位为 m/h，程序计算通常需要 m/s 或 mm/s。

rate
m/s-
mm/s-
m/h-

换算：1 m/h = 1 / 3600 m/s = 0.277777... mm/s。

转速 RPM

转速参与每转进尺和机械比能计算，需要换成 rev/s 与 rad/s。

rotation
rev/s-
rad/s-
每转时间-

换算：rev/s = rpm / 60，rad/s = 2π × rev/s。

钻头直径与井底面积

用于 WOB / A、MSE 等指标。默认按圆形井底截面积估算。

bit
直径 m-
面积 m²-
面积 mm²-

面积：A = πD² / 4。8.5 in 钻头约等于 215.9 mm。

每转进尺 DOC

DOC 由机械钻速和转速共同决定，常用于破岩效率对比。

doc
m/rev-
mm/rev-
in/rev-

DOC = ROP / rev/s。若 rpm = 0，DOC 不定义。

机械比能 MSE 估算

采用常见形式 MSE = WOB / A + 2πNT / (A × ROP)。其中 N 为 rev/s。

mse
WOB N-
Torque N·m-
MSE MPa-

MSE 只做工程估算。ROP 过低时旋转项会急剧放大，应结合工况判断数据是否有效。

常用换算关系

页面内计算统一换到 SI 单位，再输出常用工程单位。

参数关系备注
机械钻速1 m/h = 0.0002777778 m/s = 0.2777778 mm/s建模时优先用 m/s
转速1 rpm = 1 / 60 rev/s = 2π / 60 rad/sMSE 公式中 N 取 rev/s
直径1 in = 25.4 mm钻头尺寸常用英寸
钻压1 kN = 1000 N，1 tf = 9806.65 N吨力不是质量单位
扭矩1 kN·m = 1000 N·m，1 ft·lbf = 1.355817948 N·m统一到 N·m
